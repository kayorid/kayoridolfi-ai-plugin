# /kai-retro-digest

```bash
bash "${CLAUDE_PLUGIN_ROOT}/scripts/retro-digest.sh" ${ARGUMENTS:-5}
```

Mostra um resumo curto das últimas N retros. Útil em reuniões de squad ou na comunidade de Champions para captura rápida de tendências.

Para análise consolidada formal: `/kai-retro-quarterly`.
